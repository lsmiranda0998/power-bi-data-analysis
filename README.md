# Data Analysis - Power BI & Excel

## Overview

This repository contains the practical exercises and projects completed during the **Microsoft Power BI Data Analyst Professional Certificate**.

The projects cover the complete data analysis workflow, including data preparation in Microsoft Excel, data transformation, data modeling, visualization, dashboard development, business intelligence, and deployment of Power BI reports.

Throughout the course, different business scenarios were explored using Power BI and Excel to build interactive dashboards, perform data analysis, and create business reports.

## Technologies

* Microsoft Power BI
* Microsoft Excel
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* ETL (Extract, Transform and Load)
* Business Intelligence

## Running the Projects

The repository contains Power BI reports (`.pbix`), Microsoft Excel workbooks (`.xlsx`), and supporting documents used throughout the course.

To explore the projects:

* Open `.pbix` files using **Microsoft Power BI Desktop**.
* Open `.xlsx` files using **Microsoft Excel** or a compatible spreadsheet application.
* PDF and text documents can be viewed directly from the repository.
